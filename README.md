## üìö Course Overview

This course introduces students to the fundamental concepts of engineering design thinking, problem-solving methodologies, and basic web development skills. Students will learn to approach complex problems systematically while gaining practical experience with HTML and CSS.

## üéØ Learning Objectives

By the end of this course, students will be able to:
- Apply engineering design thinking to solve real-world problems
- Understand the iterative design process
- Create basic web pages using HTML and CSS
- Implement responsive design principles
- Collaborate in teams effectively on engineering projects

## üåê Web Development Learning Resources

### HTML Resources

#### **Beginner Level (Very Basic)**
- [MDN HTML Basics](https://developer.mozilla.org/en-US/docs/Learn/Getting_started_with_the_web/HTML_basics) - Official Mozilla documentation for HTML fundamentals
- [W3Schools HTML Tutorial](https://www.w3schools.com/html/) - Interactive tutorials with try-it-yourself examples
- [HTML.com Beginner's Guide](https://html.com/) - Comprehensive beginner-friendly HTML guide
- [Codecademy HTML Course](https://www.codecademy.com/learn/learn-html) - Interactive HTML learning platform
- [Khan Academy Intro to HTML](https://www.khanacademy.org/computing/computer-programming/html-css) - Video-based learning with practice exercises

#### **Intermediate Level**
- [MDN HTML Elements Reference](https://developer.mozilla.org/en-US/docs/Web/HTML/Element) - Complete reference for all HTML elements
- [HTML5 Tutorial - TutorialsPoint](https://www.tutorialspoint.com/html5/index.htm) - Comprehensive HTML5 features and semantic elements
- [FreeCodeCamp HTML Course](https://www.freecodecamp.org/learn/responsive-web-design/) - Project-based learning approach
- [HTML Dog Intermediate Tutorial](https://htmldog.com/guides/html/intermediate/) - Structured intermediate concepts
- [Web.dev Learn HTML](https://web.dev/learn/html/) - Google's comprehensive HTML course

#### **Advanced Level**
- [MDN Advanced HTML Topics](https://developer.mozilla.org/en-US/docs/Web/HTML/Global_attributes) - Advanced attributes and accessibility
- [HTML5 Boilerplate](https://html5boilerplate.com/) - Professional front-end template
- [A11y Project](https://www.a11yproject.com/) - Web accessibility best practices
- [WHATWG HTML Living Standard](https://html.spec.whatwg.org/) - Official HTML specification
- [Smashing Magazine HTML Articles](https://www.smashingmagazine.com/category/html) - Advanced techniques and best practices

### CSS Resources

#### **Beginner Level (Very Basic)**
- [MDN CSS Basics](https://developer.mozilla.org/en-US/docs/Learn/Getting_started_with_the_web/CSS_basics) - Fundamental CSS concepts
- [W3Schools CSS Tutorial](https://www.w3schools.com/css/) - Step-by-step CSS learning with examples
- [CSS-Tricks Beginner Guides](https://css-tricks.com/guides/beginner/) - Practical CSS tutorials
- [Codecademy CSS Course](https://www.codecademy.com/learn/learn-css) - Interactive CSS learning
- [Khan Academy Intro to CSS](https://www.khanacademy.org/computing/computer-programming/html-css/intro-to-css/pt/css-basics) - Visual CSS learning

#### **Intermediate Level**
- [CSS Grid Guide](https://css-tricks.com/snippets/css/complete-guide-grid/) - Complete guide to CSS Grid layout
- [Flexbox Froggy](https://flexboxfroggy.com/) - Fun game to learn CSS Flexbox
- [MDN CSS Layout](https://developer.mozilla.org/en-US/docs/Learn/CSS/CSS_layout) - Comprehensive layout techniques
- [CSS Grid Garden](https://cssgridgarden.com/) - Interactive game for learning CSS Grid
- [Responsive Web Design - FreeCodeCamp](https://www.freecodecamp.org/learn/responsive-web-design/) - Mobile-first design principles

#### **Advanced Level**
- [CSS-Tricks Advanced Techniques](https://css-tricks.com/) - Cutting-edge CSS techniques and articles
- [Smashing Magazine CSS](https://www.smashingmagazine.com/category/css/) - Advanced CSS methodologies
- [MDN CSS Animations](https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_Animations) - Advanced animations and transitions
- [Every Layout](https://every-layout.dev/) - Modern CSS layout patterns
- [CSS Specification](https://www.w3.org/Style/CSS/specs.en.html) - Official CSS specifications

### Additional Learning Tools
- [CodePen](https://codepen.io/) - Online code editor for experimenting with HTML/CSS
- [JSFiddle](https://jsfiddle.net/) - Test HTML, CSS, and JavaScript code snippets
- [Replit](https://replit.com/) - Online IDE for web development
- [VS Code](https://code.visualstudio.com/) - Recommended text editor with helpful extensions
- [Chrome DevTools](https://developers.google.com/web/tools/chrome-devtools) - Browser tools for debugging and testing
